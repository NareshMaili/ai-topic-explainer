# AI Topic Explainer

A modern web application built with **Next.js** and **Google Gemini AI** designed to help students understand complex topics through simple, easy-to-follow explanations.

## 🚀 Live Demo
[View Live Project](https://ai-topic-explainer-chi.vercel.app)

## ✨ Features
- **Student-Friendly Explanations:** Converts complex subjects into simple language using analogies.
- **Real-time AI Generation:** Powered by the `gemini-1.5-flash` model for high-speed responses.
- **Robust Error Handling:** Custom logic to handle API rate limits (429 errors) and empty inputs.
- **Responsive UI:** A clean, mobile-ready interface built with Tailwind CSS.

## 🛠️ Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **AI Integration:** Google Generative AI SDK
- **Styling:** Tailwind CSS
- **Hosting:** Vercel
